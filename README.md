## Description

[Nest](https://github.com/nestjs/nest) framework pinger repository.

## Installation

```bash
#install dependencies
$ npm install

#run the database migrations
$ npx prisma migrate dev

#run the database seeders
$ npx prisma db seed
```

## Migrations

```bash
#run the database migrations
$ npx prisma migrate dev
```

## Running the app

```bash
# dev mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
